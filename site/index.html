Regrexit Map

On 23rd June the British public voted with a majority of 51.9% to 48.1% to leave the EU. Many were shocked at the result, expressing regrexit for the outcome.

Over four million people signed a petition calling for a second chance at the referendum.

What does it all mean?

Each constituency is shaded between blue and yellow.

Blue (regrexit value between 0.0 and 1.0) indicates the population accepts the result in the constituency.

Yellow (regrexit value greater than 1.0) indicates the population regrets the result.

How is it calculated?

We compare the referendum results against the number of people who signed the petition. For each constituency we calculate the number of signatures we would expect given the remain votes cast in the area and compare it against the number of actual names on the petition. The regrexit value is then calculated as:

Regrexit = signatures / Expected.

We have to make a few assumptions, which you can find in the full details.
